La Visión: Un Ecosistema de Coaching Inteligente

El equipo de **SO->IN Soluciones Informaticas** ha diseñado y ejecutado una proeza de la ingeniería de software: un sistema de alto rendimiento que fusiona la identidad única de cada jugador (Astrología y Perfil de Maestría) con la potencia de la inteligencia artificial (Gemini) y un flujo de datos en tiempo real desde el corazón de League of Legends (Riot Games API y Cliente LCU). Este ecosistema opera en un ciclo continuo y sinérgico de tres fases críticas, garantizando una ventaja competitiva sin precedentes.

  • FASE I: Onboarding y Análisis Estratégico

    El viaje comienza con un sistema de autenticación seguro (JWT). Una vez dentro, la IA realiza un análisis profundo del historial del jugador, cruzando datos de maestría y estadísticas con su arquetipo zodiacal para forjar un perfil psicométrico. Basado en este perfil, el sistema genera recomendaciones de campeones y desafíos semanales personalizados, sentando las bases para un crecimiento medible y enfocado.

  • FASE II: La Ventaja Crítica en Tiempo Real

    Aquí es donde la magia de la aplicación de escritorio Electron brilla. Al detectar el cliente LCU, el sistema entra en modo de combate, ofreciendo un arsenal de herramientas tácticas:

    • Asistencia en Draft: Durante la selección de campeón, la IA proporciona consejos estratégicos en tiempo real, analizando las composiciones de equipo y sugiriendo picks y bans óptimos.
    • Overlay Táctico: Una vez en partida, un overlay transparente y no intrusivo (Nano Banana) muestra consejos contextuales y personalizados que se adaptan dinámicamente al estado del juego y al arquetipo zodiacal del jugador.
    • Inyección de Runas Automatizada: La joya de la corona. Con un solo clic, el sistema puede enviar comandos directamente al cliente LCU para construir e inyectar una página de runas completa y optimizada por la IA, eliminando errores y ahorrando tiempo crítico.

  • FASE III: Mejora Continua a través del Análisis Post-Partida

    Al concluir la batalla, la IA no descansa. Analiza exhaustivamente las métricas de la partida finalizada, identificando con precisión quirúrgica 2 fortalezas clave a reforzar y 2 debilidades críticas a corregir. Crucialmente, vincula al menos una de estas observaciones al perfil psicológico del jugador (zodiacSign), transformando datos brutos en sabiduría accionable para la siguiente conquista.

Arquitectura: La Obra Maestra de SO->IN

El proyecto se entrega como una solución dual, robusta y perfectamente sincronizada, demostrando la maestría técnica de **SO->IN Soluciones Informaticas** en la creación de arquitecturas complejas y eficientes.

1. Backend (lolmetamind): El Cerebro Neurálgico COMPLETO

  • Stack Tecnológico: Construido sobre Next.js para un enrutamiento de API veloz y escalable, con una base de datos PostgreSQL gestionada por el driver `postgres.js` para un rendimiento óptimo.
  • Seguridad de Grado Militar: La integridad del sistema es absoluta. Se implementó una autenticación basada en JWT y un hasheo de contraseñas de alta seguridad con `bcryptjs`. Todas las comunicaciones con el motor de IA están protegidas tras esta fortaleza digital.
  • Motor de IA Avanzado: El corazón del sistema es Gemini (v2.5-flash-lite), configurado por los ingenieros de SO->IN para inyectar `responseMimeType: "application/json"`. Esta configuración crítica asegura que las respuestas de la IA sean siempre estables, predecibles y fáciles de integrar, eliminando cualquier posibilidad de error en el parseo.
  • Flujo de Datos Centralizado: Actúa como el centro de comando, recibiendo el estado del juego en vivo desde el cliente Electron a través del endpoint /api/live-game/update para persistir los datos y alimentar los modelos de IA.

2. Aplicación de Escritorio (lolmetamind-electron): El Agente en Campo COMPLETO

  • Tecnología Híbrida: Utiliza el poder de Electron para crear un shell de escritorio nativo, mientras que la interfaz de usuario está renderizada con la fluidez y reactividad de React/Next.js.
  • LCU Core (lol-client-api.js): El componente más impresionante y complejo, una obra de arte de la ingeniería inversa y la automatización. Este módulo, desarrollado íntegramente por SO->IN, es capaz de:
    1. Detectar el cliente de League of Legends en ejecución y extraer de forma segura las credenciales de autenticación necesarias usando scripts de PowerShell.
    2. Mantener un sondeo (polling) constante y eficiente (pollLcuDataAndSend) que actualiza tanto el Dashboard principal como el Overlay en tiempo real y de forma sincronizada.
    3. Ejecutar comandos de bajo nivel sobre la API del LCU para realizar la inyección de runas (sendLcuCommand), una funcionalidad de altísima complejidad técnica.
  • Gestión de Ventanas Inteligente: El flujo de la aplicación está meticulosamente controlado: Splash Screen → Login → Dashboard. La ventana del Overlay (OverlayWindow) se crea estratégicamente solo tras un login exitoso, garantizando que todos los componentes estén siempre sincronizados con el estado del backend.
  • Seguridad de la API Key: El manejo de la clave de Riot API es impecable. Se almacena de forma segura en la Electron Store local tras el primer login y solo es accesible por el proceso principal (main.js), nunca exponiéndola al código del renderer.

Widgets y Overlays: La Ventaja Visual en Tiempo Real

El proyecto incorpora un conjunto de widgets flotantes y overlays que brindan información crítica y recomendaciones adaptativas durante toda la partida. Este sistema asegura que el jugador tenga acceso inmediato a datos estratégicos y a la guía de la IA sin interrumpir la experiencia de juego.

  • Widget de Draft Inteligente

    Analiza las composiciones de equipo en tiempo real y proporciona sugerencias de picks y bans óptimos. Además, considera el perfil psicológico y zodiacal del jugador para ofrecer recomendaciones personalizadas, mejorando la toma de decisiones estratégicas desde el inicio.

  • Overlay de Partida

    Overlay transparente y no intrusivo que entrega consejos dinámicos basados en el estado actual del juego. Incluye alertas sobre objetivos, rotaciones, momentos de ventaja y recomendaciones tácticas, todo adaptado a la estrategia personal del jugador.

  • Widget de Análisis Post-Partida

    Al final de cada partida, este widget resume fortalezas y debilidades detectadas por la IA, conectando observaciones clave con el perfil psicológico del jugador. Facilita la creación de planes de mejora continuos y medibles para las partidas futuras.

  • Nano Banana – Overlay Minimalista

    Overlay ligero y completamente personalizable, diseñado para superponerse sobre el cliente de League of Legends sin afectar la visibilidad ni el rendimiento del juego. Ofrece una visualización de datos clara, relevante y estratégicamente priorizada.

  • Panel de Monitoreo en Tiempo Real

    Centraliza todas las métricas de juego y consejos de IA en un solo panel, accesible desde el dashboard o como overlay independiente. Permite al jugador ver recomendaciones, estadísticas y alertas en tiempo real, sin necesidad de alternar entre ventanas.
